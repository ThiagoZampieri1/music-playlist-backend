Webservice - Playlists

Rotas para Usuários

GET /usuarios

Lista todos os usuários.

POST /usuarios

Criar um novo usuário.

  • nome: Nome do usuário (obrigatório).
  • email: Email do usuário (obrigatório).
  • senha: Senha do usuário (obrigatório).

Rotas para Playlists

GET /playlists

Lista todas as playlists.

POST /playlists

Cria uma nova playlist.

  • titulo: Título da playlist (obrigatório).
  • usuario_id: ID do usuário criador da playlist (obrigatório).

Rotas para Músicas

GET /musicas?playlist_id={id}

Lista todas as músicas de uma playlist específica.

POST /musicas

Adiciona uma nova música a uma playlist.

  • titulo: Título da música (obrigatório).
  • link: Link da música (obrigatório).
